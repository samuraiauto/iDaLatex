# iDaLatex - Витрина + продажа цифровых товаров

Статический сайт + небольшой Node.js бэкенд для приема оплаты (СБП через провайдера) и выдачи цифровых товаров на email через API поставщика (B2B).

## Особенности

- 🎨 Современный и красивый дизайн
- 📱 Полностью адаптивный (responsive) дизайн
- 🛒 Функциональная корзина покупок
- 🔍 Фильтрация продуктов по категориям
- 💾 Сохранение корзины в localStorage
- ⚡ Плавная анимация и переходы
- 📋 Детальная информация о продуктах

## Структура проекта

```
iDaLatex/
```

## Загрузка на GitHub

Проект уже инициализирован как git репозиторий. Для загрузки на GitHub:

### Вариант 1: Использование скрипта (рекомендуется)

1. Создайте новый репозиторий на GitHub: https://github.com/new
2. Скопируйте URL вашего репозитория (например: `https://github.com/username/iDaLatex.git`)
3. Запустите скрипт:
```bash
./deploy-to-github.sh https://github.com/username/iDaLatex.git
```

### Вариант 2: Ручная загрузка

1. Создайте новый репозиторий на GitHub: https://github.com/new
2. Выполните команды:
```bash
git remote add origin https://github.com/username/iDaLatex.git
git branch -M main
git push -u origin main
```

## Запуск проекта

### Локально (фронт + бэкенд вместе)

1. Установите зависимости бэкенда:

```bash
cd backend
npm i
```

2. Создайте `.env` на основе примера:

```bash
cp .env.example .env
```

3. Запустите сервер:

```bash
npm run dev
```

4. Откройте сайт: `http://localhost:3000`

### Вебхуки

ЮKassa должна уметь отправлять вебхук на:

- `POST /api/yookassa/webhook`

После события `payment.succeeded` бэкенд вызывает API поставщика и отправляет коды на email.

## Кастомизация

### Изменение цветовой схемы
Отредактируйте CSS переменные в файле `styles.css`:

```css
:root {
    --primary-color: #2d5016;
    --primary-light: #4a7c2a;
    --secondary-color: #f5f5f0;
    --accent-color: #8b9a5b;
    /* ... */
}
```

### Добавление новых продуктов
Добавьте объект продукта в массив `products` в файле `products.js`:

```javascript
{
    id: 15,
    name: "Название продукта",
    category: "pillows", // или "mattresses"
    price: 5000,
    description: "Описание продукта",
    features: ["Характеристика 1", "Характеристика 2"],
    emoji: "🛏️"
}
```
